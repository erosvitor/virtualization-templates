## About
Vagrant virtual machines.

## Vagrant settings
- Install VirtualBox
- Install Vagrant

## Vagrant virtual machines

### Jenkins with JDK 17
1. Run virtual machine
```
cd jenkins-java17
vagrant up
```

2. Access Jenkins
```
http://localhost:8080
```

### Nexus
1. Run virtual machine
```
cd nexus
vagrant up
```

2. Access Nexus
```
http://localhost:8081
```
